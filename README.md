# 🐶 Canine Heart Disease Analysis Using Data Mining Techniques
Data mining-based analysis of canine heart disease using ECG data, leveraging K-Means clustering and Isolation Forest to identify cardiac risk patterns through HRV and R-R interval features for early detection and healthcare insights.


## 📖 Project Overview
This project focuses on discovering **risk patterns in canine heart disease** using data mining techniques on ECG and physiological data. By analyzing features such as **Heart Rate Variability (HRV)** and **R-R intervals**, the system identifies abnormal cardiac behaviors and potential health risks to support early diagnosis.

---

## 🎯 Objectives
- Discover hidden risk patterns in canine heart disease data  
- Analyze ECG signals using HRV and R-R interval features  
- Apply clustering and anomaly detection techniques  
- Support early detection of cardiac abnormalities  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib  
- **Techniques:** K-Means Clustering, Isolation Forest, Data Mining  

---

## ⚙️ Methodology
1. Data Collection (ECG & physiological dataset)  
2. Data Preprocessing and Cleaning  
3. Feature Extraction (HRV, R-R intervals)  
4. Clustering using K-Means  
5. Anomaly Detection using Isolation Forest  
6. Risk Pattern Identification  
7. Visualization of results  

---

## 📊 Results
- Identified distinct clusters representing cardiac behavior patterns  
- Detected abnormal heart conditions using anomaly detection  
- Discovered potential risk patterns in canine heart health

## 🌍 Impact

- Assists veterinarians in clinical decision-making  
- Demonstrates application of data mining in healthcare analytics
